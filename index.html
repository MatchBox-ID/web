    01. Apa yang dimaksud dengan hosting?

    Hosting atau web hosting adalah penggunaan server untuk menyimpan file dan database website sehingga bisa diakses di internet. Agar berfungsi dengan baik, website memerlukan paket hosting dengan resource yang cukup untuk menyimpan semua data.

    02. Bagaimana cara menghubungkan domain dan hosting?

    Pertama-tama, ganti informasi nameserver di MPanel. Setelah itu, arahkan domain ke nameserver Matchbox.
